Basic syntax

CSS is made up of selectors and semicolon-separated lists of declarations.

Each declaration is made up of a property-value pair.

Selectors determine which part of the web page to style; declarations define how to style them.

Do More with Selectors

If two groups of elements share some style declarations, group selectors together as a comma-separated list

Chain selectors as a list without any separation to select an element that has both selectors

Select elements that are nested inside other elements by adding a single space between selectors

Every single thing on a webpage is a rectangular box. These boxes can have other boxes in them and can sit alongside one another. Laying out a webpage is all about deciding how you are going to nest and stack these boxes.
